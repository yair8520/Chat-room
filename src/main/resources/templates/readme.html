Execution

First the user will enter url , then "Validate" servlet will validate the url ,
if the url is ok then we will continue to "startCrawling" servlet,
if not the error page message will e displayed,
"StartCrawling" servlet will open a session to recognize the user,
and will start a thread that will be saved in servlet context.
"Redirect" servlet will display a redirect page that which announces the beginning of crawling
"result" servlet will display the result page and will recognize the user

All pages are protected
